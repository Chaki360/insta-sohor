JavaScript is something that we called a Client-side Scripting Language which means it has the ability to run inside an browser. Mainly Javascript works with HTML & CSS to build application.We can design dynamic interactive website using our javascript code. Javascript we can use to validate data.For example our login password validation. How javascript works?Well our web browser run a webpage from internet then parse the HTML and creates Documents Object Model (DOM) from the clients and DOM gives us the live view of the web page from Javascript code.Then browser executives data which parsed from CSS & HTML file.After that the browser javascript engine loads javascript files.Then javascript starts to executive itself synchronously.That's how it works.

Session Storage is a kind of alternative to cookies.Its very useful when we want to store & use data on browser.On the other hand Local Storage allows web developers to store and also retrieve data in browser and this data doesn't expire like the session storage. That means all the data will be saved even if the tab or the browser is closed while session storage doesn't save any of the data after closing the browser or even that tab.
